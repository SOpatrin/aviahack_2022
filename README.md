## Instruction for local start

### Requirements

- node >=16

### Build

You need to clone repo, then enter the following commands:

Move to cloned repo

`cd aviahack_2022`

Install dependencies

`npm install`

Build project

`npm run build`

Start local server

`npm run start`

Open [http://localhost:3000](http://localhost:3000) you would see project localy
